Ejercicio Práctico: Uso de Imágenes sin CSS

Objetivo

Aprender a insertar imágenes en una página web utilizando únicamente HTML.

Instrucciones

Debes crear una página web que incluya:

  1. Un título principal.
  2. Una imagen local.
  3. Una imagen desde internet.
  4. Texto alternativo usando alt.
  5. Ancho y alto definidos.
  6. Una imagen que funcione como enlace.
  7. Uso de figure y figcaption. (no se hace)
  8. Uso de loading="lazy". (no se hace)
  9. No utilizar CSS.

Actividad

Completa el ejercicio agregando:

  • Dos imagenes adicionales

  • Una imagen en formato WebP o SVG.

  • Un enlace a otra página usando una imagen.

Preguntas

  1. ¿Qué función cumple el atributo alt?
    2. Proporciona una descripción textual de una imagen cuando la misma esta rota por x motivo.

  2. ¿Qué ventaja tiene loading="lazy"?
    3. Mejora principalmente la velocidad de carga inicial de la página

  3. ¿Qué diferencia existe entre PNG y JPG?
    4. Depende de compresión y la transparencia, este último es png

  4. ¿Para qué sirve figcaption?
    5. Sirve para definir el título o leyenda (pie de foto).
